### 1. Search and Availability

**As a camper,**

I want to view a calendar showing available dates for specific campsites,

So that I can plan my trip based on when the site is actually free.

#### Acceptance Criteria:

- Calendar displays availbility slots.
- Unavailable dates are "greyed out" or unclickable.
- Users can select a start and end date.

### 2. Secure Payment

**As a camper,**

I want to pay for my reservation via credit card or digital wallet within the app,

So that my booking is instantly confirmed without needing to call the office.

#### Acceptance Criteria:

- Integration with a payment processor (e.g., Stripe, PayPal).
- Users receive a digital receipt upon successful transaction.
- Encryption is used to handle sensitive data.

### 3. Booking Management

**As a campsite manager,**

I want to view, edit, or cancel customer bookings in a central dashboard,

So that I can keep track of occupancy and handle customer requests manually if needed.

#### Acceptance Criteria:

- A searchable list of all current and upcoming bookings.
- Ability to change booking dates or status (e.g., "Checked In").
- Automatic updates to the public availability calendar when a change is made.
